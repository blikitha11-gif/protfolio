# 💼 Personal Portfolio Website

A modern and responsive personal portfolio website built to showcase my education, technical skills, internship experience, certifications, achievements, and projects in a clean and elegant interface.

This portfolio serves as my professional digital presence and highlights my journey as a BCA Data Science student and aspiring Software Developer.

---

## 🌐 Live Demo

https://blikitha11-gif.github.io/protfolio/

---

## 📌 Features

- Responsive design for desktop, tablet, and mobile
- Elegant Hero section
- About Me
- Education
- Internship Experience
- Technical Skills
- Projects Showcase
- Certifications
- Achievements
- Contact Section
- Smooth scrolling navigation
- Scroll animations using Intersection Observer
- Clean and minimal UI
